1. To run this notebook you should first create and activate an enviroment:
		- __Linux__ or __Mac__: 
	```
	conda create -n maia_aia python=3.6
	source activate maia_aia
	```
	- __Windows__: 
	```
	conda create --name maia_aia python=3.6
	activate maia_aia
	

2.  Install a few required pip packages, which are specified in the requirements text file (including OpenCV).

	pip install -r requirements.txt

3. To execute the jupyter you should change the path of the folders, and create a folder to save the results.
